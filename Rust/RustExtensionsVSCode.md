# Rust Extensions for easier development in VSCode

I installed these Rust extensions into my VSCode:

- [Rust-analyzer](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer)

- [CodeLLDB: A debugger for C++, Rust, & Other Compiled Languages](https://github.com/vadimcn/codelldb) 

- [CodeLLDB: Manual](https://github.com/vadimcn/codelldb/blob/v1.10.0/MANUAL.md)

- [Even Better TOML: A TOML language support extension - bc TOML is deprecated](https://marketplace.visualstudio.com/items?itemName=tamasfe.even-better-toml)

- [Dependi - bc Crates is deprecated: A dependency management extension for Rust.](https://github.com/filllabs/dependi)


### Resource that suggested these extensions

- [Rust 101 - How to set up VS Code for Rust](https://youtu.be/Mfv8zFzDFvI?si=zKVyl_ePQo9BtBh1)
